# ­ЪД« CNF Resolution Visualizer
  рЃћрЃА рЃљрЃарЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ Python-рЃўрЃАрЃљ рЃЊрЃљ PyQt5-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃЏрЃўрЃАрЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў CNF рЃцрЃЮрЃарЃЏрЃБрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃЊрЃњрЃўрЃюрЃЮрЃА, рЃљрЃарЃўрЃљрЃю рЃЌрЃБ рЃљрЃарЃљ рЃўрЃАрЃўрЃюрЃў рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃЮрЃЉрЃарЃўрЃЋрЃюрЃў рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.
  рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃў рЃљрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃърЃарЃћрЃЊрЃўрЃЎрЃљрЃбрЃБрЃџ рЃџрЃЮрЃњрЃўрЃЎрЃљрЃерЃў CNF рЃцрЃЮрЃарЃЏрЃБрЃџрЃћрЃЉрЃўрЃА рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃЮрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЊрЃњрЃћрЃюрЃљрЃЊ. рЃўрЃњрЃў рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃљрЃЊ рЃБрЃЊрЃћрЃЋрЃА рЃЉрЃћрЃЋрЃарЃў SAT-solver-рЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ:
 рЃерЃћрЃЋрЃўрЃбрЃљрЃюрЃЮрЃЌ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЎрЃџрЃљрЃБрЃАрЃћрЃЉрЃў CNF рЃцрЃЮрЃарЃЏрЃљрЃерЃў.
 рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃўрЃЌ рЃерЃћрЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃЌ, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃљрЃЌрЃў рЃЊрЃљрЃЎрЃЏрЃљрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљ.
 рЃЏрЃўрЃЋрЃўрЃдрЃЮрЃЌ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃдрЃгрЃћрЃарЃљ рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.
 рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌ, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃцрЃЮрЃарЃЏрЃБрЃџрЃљ:
    рЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃЮрЃЉрЃарЃўрЃЋрЃў (UNSAT)
    рЃЊрЃљрЃЎрЃЏрЃљрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљрЃЊрЃў (SAT)
